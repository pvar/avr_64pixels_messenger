Easy 8x8 Sprite Maker
by Draco_X5 (John Stinnett)
Released: June 18, 2003
Console: Windows

This is a quick program I made while learning how to use Borland Delphi 7. Simply use the
check boxes as a "bitmap" of sorts to draw your picture. Then click on either of the copy
buttons to send the data to the Windows clipboard. It can then be pasted in any kind of
editor. Have fun!!!
